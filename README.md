Munny pouches give random amount of munny between 500 and 3000. The specific amount given is printed in the console, which can be viewed by pressing F2.

Enemies never drop munny (Gambler reaction command game is unaffected by this).

One free munny pouch is in the Proof of Nonexistence chest in GoA.

Randomizer settings suggestions

- disable "Garden of Assemblage" checks
  - this mod replaces one chest with a Munny Pouch
- enable "Remove Non-Superboss Popups"
  - the munny pouches are set to popups so will overwrite any item on them. this setting sets them to junk, except for superbosses.
- add maximum amount of Visit Unlocks to Shop and enable Keyblades in shop
  - set prices to whatever you think is fair. suggested: Visit Unlocks around 500 and Keyblades around 1000
- enable other Shop Items
- set "Junk Items" to consumables that can be sold
- enable "Keyblades Unlock Chests"
- for more difficulty
  - use Path Hints
  - set "Proof Depth" to "Superbosses"
  - start with no Visit Unlocks
  - enable Synthesis
